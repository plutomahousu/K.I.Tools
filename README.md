K.I.Tools
=========

- KIT学生にとって便利なChrome拡張機能集

####拡張機能一覧
- **[自動ログイン]()**
    - 学生ポータル，VPN，給与明細照会，見積依頼に自動ログイン
- **[カレンダー同期]()**
    - 大学のカレンダーをグーグルカレンダーに同期
- **[AmaLin]()**
    - Amazonで本を検索した際、ライブラリーセンターの蔵書状態を表示
- **[シャトルバス]()**
    - 扇が丘と八束穂を結ぶシャトルバスの時刻表閲覧・通知
